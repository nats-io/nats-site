Support

Reporting Bugs

If you'd like to report a bug/defect, please open a GitHub Issue on the relevant NATS repository. We've provided some guidelines for doing so here.

Community Support for NATS and Clients

Often the quickest way to get help is to start a discussion with other NATS users. There are several public forums where you can discuss NATS with other knowledgeable users.
